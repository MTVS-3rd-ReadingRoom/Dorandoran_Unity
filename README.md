# Minecraft
매타버스아카데미 2차 융합 비게임 창작 프로젝트 

독서토론 플랫폼 도란도란

## 프로젝트 소개
기획 1명, 유니티 2명, 백엔드 3명 AI 1명으로 구성된 팀이 협업하여 진행한 비게임 프로젝트

Photon Pun을 사용해 멀티환경 구축과 Http 통신으로 백엔드와 연결을 목표
